## 🚀 Quick Start

### Prerequisites

- **Node.js** 18+ 
- **npm** or **yarn**

### Setup

```bash
# Clone the repository
git clone https://github.com/kumarharsh21112003/Portfolio.git

# Navigate to project
cd Portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the magic ✨

### Production Build

```bash
npm run build
npm start
```

## ⚙️ Customization

<details>
<summary><b>📝 Update Personal Info</b></summary>

Edit `lib/constants.ts` to update:

- `SOCIAL_LINKS` — Phone, LinkedIn, etc.
- `DEV_PROJECTS` — Your project showcase
- `SKILL_CATEGORIES` — Your tech skills
- `EXPERIENCE` — Work experience entries
- `EDUCATION` — Academic background
- `EQUIPMENT` — Your setup/gear

</details>

<details>
<summary><b>🖼️ Update Images</b></summary>

1. Add images to `public/assets/images/`
2. Update paths in `lib/constants.ts` or respective components
3. Profile image: update `Hero.tsx` and `ProfileCard.tsx`

</details>

<details>
<summary><b>📄 Update Resume</b></summary>

Replace `public/assets/kumarharshresume.pdf` with your updated resume.

</details>
